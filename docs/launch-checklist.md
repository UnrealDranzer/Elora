# Launch Checklist

- Purchase and configure:
  - `www.elora.com`
  - `admin.elora.com`
  - `api.elora.com`
- Run full regression testing on a staging environment.
- Verify Razorpay live mode with low-value real transactions.
- Configure Cloudinary delivery transforms and usage alerts.
- Seed a real admin user and disable default seed credentials.
- Add real product images, banners, categories, collections, and policies.
- Confirm tax, invoicing, shipping, and COD policies with operations.
- Set up database backups and point-in-time recovery.
- Verify log drains, uptime monitoring, and alerting.
- Confirm legal pages, privacy policy, refund policy, and TOS are published.
